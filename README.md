# Track Bilibili followers V2

## Switched to deno and ts

## Usage

[Download deno](https://deno.land/)

- directly run the ts file

```shell
./mod.ts -u $uuid -i $interval
```

you can also run

```shell
./lex.sh
```

## Install:
```shell
./build.sh
```
will defaultly generate an executable at `./bin`
